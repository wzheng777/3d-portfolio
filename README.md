# 3D Portfolio Demo

This is a demo application showcasing the power of **React** and **Three.js** (via `@react-three/fiber` and `@react-three/drei`) to create immersive 3D web experiences.

## Features
- Interactive 3D scene
- Built with React 19 and Vite
- Styled with Tailwind CSS (if applicable) or custom CSS

## Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/wzheng777/3d-portfolio.git
   cd 3d-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` (or the port shown in your terminal).

## Deployment

This project is configured for deployment to GitHub Pages.

To deploy a new version:
```bash
npm run deploy
```
This command will build the project and push it to the `gh-pages` branch.
